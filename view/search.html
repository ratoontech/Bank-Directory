Bank Directory API

This API provides you with the INDIAN bank details. You can perform searching bank details with various fields like IFSC, MICR, Bank name, Branch etc.


Endpoints

/search?bank={bank}&branch={branch} - Retrieves the bank details with matching bank and branch.

/search/ifsc/{ifsc} - Retrieves the bank details with matching IFSC.

/search/micr/{micr} - Retrieves the bank details with matching MICR.

/search/bank/{bank} - Retrieves the bank details with partial/full matching bank name.

/search/branch/{branch} - Retrieves the bank details with partial/full matching branch name.

/search/state/{state} - Retrieves the bank details with partial/full matching state name.

/search/district/{district} - Retrieves the bank details with partial/full matching district name.

/search/city/{city} - Retrieves the bank details with partial/full matching city name.
